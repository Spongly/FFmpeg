FFmpeg Commands

Convert video to another format

ffmpeg -i input.mp4 output.mov

Trim video

ffmpeg -ss 00:00:10 -i input.mp4 -t 00:00:30 -c:v copy -c:a copy output.mp4

Extract audio from video

ffmpeg -i input.mp4 -vn -acodec copy output.aac

Resize video

ffmpeg -i input.mp4 -vf "scale=640:480" output.mp4

Concatenate multiple videos

ffmpeg -i input1.mp4 -i input2.mp4 -filter_complex "concat=n=2:v=1:a=1" output.mp4

Add watermark to video

ffmpeg -i input.mp4 -i watermark.png -filter_complex "overlay=10:10" output.mp4

Extract frames from video

ffmpeg -i input.mp4 -vf "fps=1" output%d.png

Convert video to animated GIF

ffmpeg -i input.mp4 -vf "fps=10,scale=320:-1:flags=lanczos" -c:v gif output.gif

Extract specific segment from video

ffmpeg -ss 00:00:30 -i input.mp4 -t 00:00:10 -c:v copy -c:a copy output.mp4

Convert video to audio

ffmpeg -i input.mp4 -vn -acodec copy output.mp3

Add subtitles to video

ffmpeg -i input.mp4 -i subtitles.srt -c:v copy -c:a copy -c:s mov_text output.mp4

Merge audio and video

ffmpeg -i input.mp4 -i audio.mp3 -c:v copy -c:a aac -map 0:v:0 -map 1:a:0 output.mp4

Change video speed

ffmpeg -i input.mp4 -vf "setpts=0.5*PTS" output.mp4

Create a video slideshow from images

ffmpeg -framerate 1/5 -i img%d.jpg -c:v libx264 -r 30 -pix_fmt yuv420p output.mp4

Extract metadata from video

ffprobe -v quiet -print_format json -show_format -show_streams input.mp4

Create a video with a black screen

ffmpeg -f lavfi -i color=c=black:s=1280x720:d=10 -vf "drawtext=text='Cicada 3301':fontcolor=white:fontsize=48:x=(w-text_w)/2:y=(h-text_h)/2" output.mp4

Convert video to grayscale

ffmpeg -i input.mp4 -vf "format=gray" output.mp4

Rotate video

ffmpeg -i input.mp4 -vf "transpose=1" output.mp4

Extract specific audio streams from video

ffmpeg -i input.mp4 -map 0:a:1 -c copy output.aac

Reverse video

ffmpeg -i input.mp4 -vf "reverse" output.mp4
